# Roadmap

## Stage 0: Project Setup

Create repositories, service skeletons, frontend structure, infrastructure folders, and project documentation.

## Stage 1: Auth Service

Build signup, login, password hashing, JWT issuance, role claims, and protected `me` endpoint.

## Stage 2: Athlete Service

Build athlete CRUD, profile fields, goals, position, injury notes, coach-athlete relationship, and timeline records.

## Stage 3: Frontend Dashboard

Build login page, protected app shell, dashboard navigation, athlete list, and athlete profile pages.

## Stage 4: Media Service

Build practice sessions, signed upload URL generation, upload completion, video metadata, and athlete video listing.

## Stage 5: Timeline

Unify athlete profile changes, videos, AI reviews, coach approvals, and drill assignments into a chronological timeline.

## Stage 6: AI Review

Generate review summaries, observations, strengths, improvement areas, and recommended drills from video context.

## Stage 7: Coach Review Workflow

Allow coaches to edit AI output, approve reviews, reject drafts, and decide athlete visibility.

## Stage 8: Drill Assignment

Create drill library records and assign drills to athletes from coach review recommendations.

## Stage 9: Athlete Dashboard

Build athlete-facing views for feedback, assigned drills, timeline, and progress status.

## Stage 10: Progress Insights

Add trend summaries, completion metrics, recurring improvement areas, and coach-facing insights.

## Stage 11: Deployment

Prepare Docker Compose, CI/CD, production environment variables, database migrations, logging, and monitoring.
